## Overview

This quick start guide will help you conduct your first risk assessment with BioRAM. Whether you're evaluating biosafety, biosecurity, or chemical security risks, the basic workflow is the same.

<Note>
Before you begin, make sure you've [downloaded and installed BioRAM](/installation). This guide assumes you have either the Excel or Java version ready to use.
</Note>

## Select your assessment type

Determine which type of risk assessment you need to perform:

<Tabs>
  <Tab title="Biosafety">
    **Use for:** Evaluating biological safety risks in laboratories handling infectious agents or potentially hazardous biological materials.
    
    **Key focus areas:**
    - Agent pathogenicity and characteristics
    - Laboratory containment measures
    - Personnel training and procedures
    - Exposure potential and mitigation
    
    **Files to use:**
    - Excel: `BioRAM-Safety-2016.xlsx` or `BioRAM-2022 (BioRisk).xlsx`
    - Java: `Biosafety.raml` with `bioram-4.0.5.jar`
    
    [Learn more about biosafety assessment →](/assessments/biosafety)
  </Tab>
  
  <Tab title="Biosecurity">
    **Use for:** Assessing security measures for biological materials that could pose dual-use concerns or security risks.
    
    **Key focus areas:**
    - Access control and physical security
    - Personnel reliability and screening
    - Inventory management
    - Threat scenarios and vulnerabilities
    
    **Files to use:**
    - Excel: `BioRAM-Security-2020.xlsx` or `BioRAM-2022 (BioRisk).xlsx`
    - Java: `Biosecurity.raml` with `bioram-4.0.5.jar`
    
    [Learn more about biosecurity assessment →](/assessments/biosecurity)
  </Tab>
  
  <Tab title="Integrated BioRisk">
    **Use for:** Comprehensive assessment combining both safety and security considerations in a unified framework.
    
    **Key benefits:**
    - Holistic risk evaluation
    - Identifies synergies between safety and security measures
    - Streamlined assessment process
    - Optimized resource allocation
    
    **Files to use:**
    - Excel: `BioRAM-2022 (BioRisk).xlsx` (recommended)
    - Java: Combined assessment using appropriate `.raml` files
    
    [Learn more about integrated biorisk →](/assessments/integrated-biorisk)
  </Tab>
  
  <Tab title="Chemical Security">
    **Use for:** Evaluating security risks associated with hazardous chemicals that could be misused or diverted.
    
    **Key focus areas:**
    - Chemical inventory and tracking
    - Facility security measures
    - Personnel access controls
    - Theft and diversion prevention
    
    **Files to use:**
    - Java: `ChemSAM.raml` with `bioram-4.0.5.jar`
    
    [Learn more about ChemSAM →](/guides/chemsam)
  </Tab>
</Tabs>

## Complete your first assessment

Follow these steps to conduct a basic risk assessment:

<Steps>
  <Step title="Open the tool and provide facility information">
    **Excel version:**
    1. Open your chosen `.xlsx` file in Microsoft Excel
    2. Click **Enable Content** to allow macros
    3. Navigate to the first worksheet (usually "Facility Information" or "Introduction")
    4. Enter basic information about your laboratory facility
    
    **Java version:**
    1. Launch the application (double-click `.jar` or `.exe`)
    2. Select **File → New Assessment**
    3. Choose the appropriate `.raml` file for your assessment type
    4. Complete the facility information section
    
    <Tip>
    Have the following information ready:
    - Facility name and location
    - Laboratory biosafety level (BSL-1, BSL-2, BSL-3, or BSL-4)
    - Types of agents or chemicals handled
    - Number of personnel with access
    </Tip>
  </Step>
  
  <Step title="Answer assessment questions">
    Work through the structured questionnaire, answering questions about:
    
    **For biosafety assessments:**
    - Agent characteristics (pathogenicity, mode of transmission)
    - Laboratory design and engineering controls
    - Standard operating procedures
    - Personal protective equipment
    - Personnel training and competency
    - Emergency response capabilities
    
    **For biosecurity assessments:**
    - Physical security barriers and controls
    - Access control systems and procedures
    - Personnel screening and reliability programs
    - Inventory management and accountability
    - Cybersecurity measures
    - Incident detection and response
    
    <Note>
    Answer questions based on your current practices, not idealized scenarios. Honest assessment is critical for identifying real risks and effective mitigation strategies.
    </Note>
    
    **Question format:**
    - Most questions use multiple-choice or yes/no formats
    - Some require numeric inputs (e.g., number of personnel, facility size)
    - Select the option that best describes your current situation
    - Use "Not Applicable" or "Unknown" options when appropriate
    
    <Accordion title="What if I don't know an answer?">
      If you're uncertain about an answer:
      1. Consult with relevant personnel (facility managers, safety officers, security personnel)
      2. Review facility documentation and standard operating procedures
      3. Conduct a brief site inspection if needed
      4. Use the "Unknown" option temporarily and flag the question for follow-up
      5. Consider this an opportunity to identify gaps in your facility's documentation
    </Accordion>
  </Step>
  
  <Step title="Review risk scores and results">
    Once you've completed the questionnaire, the tool automatically calculates risk scores.
    
    **Understanding your results:**
    
    BioRAM typically provides risk scores on a three-tier scale:
    - **Low risk**: Current measures are adequate; maintain existing controls
    - **Medium risk**: Additional mitigation recommended; prioritize based on resources
    - **High risk**: Immediate action required; implement additional controls promptly
    
    **What to review:**
    1. **Overall risk score**: Summary assessment of your facility's risk level
    2. **Risk by category**: Breakdown by specific risk factors (e.g., containment, procedures, security)
    3. **Heat maps or risk matrices**: Visual representation of risk distribution
    4. **Comparison to baselines**: How your facility compares to recommended practices
    
    <Accordion title="View example risk output">
      ```
      Overall Risk Assessment: MEDIUM
      
      Risk by Category:
      - Agent Hazard Level: HIGH
      - Containment Measures: MEDIUM
      - Personnel Training: LOW
      - Physical Security: MEDIUM
      - Access Control: HIGH
      
      Priority Actions:
      1. Enhance access control procedures
      2. Review containment equipment maintenance
      3. Implement additional security cameras in storage areas
      ```
    </Accordion>
  </Step>
  
  <Step title="Identify mitigation measures">
    Based on your risk scores, the tool recommends specific mitigation measures:
    
    **Types of recommendations:**
    - **Engineering controls**: Physical modifications to enhance safety or security
    - **Administrative controls**: Policy and procedure improvements
    - **PPE recommendations**: Personal protective equipment enhancements
    - **Training needs**: Identified gaps in personnel competency
    - **Security enhancements**: Physical or procedural security improvements
    
    **Prioritizing actions:**
    1. Address high-risk areas first
    2. Consider cost-effectiveness and feasibility
    3. Identify quick wins vs. long-term projects
    4. Involve relevant stakeholders (management, safety committee, security personnel)
    
    <Tip>
    Create an action plan with:
    - Specific mitigation measures to implement
    - Responsible personnel for each action
    - Target completion dates
    - Budget estimates if applicable
    - Success metrics to verify effectiveness
    </Tip>
  </Step>
  
  <Step title="Save and export your assessment">
    **Excel version:**
    - Save the file with a descriptive name (e.g., `BioRAM-MyLab-2024-03.xlsx`)
    - Export results to PDF if needed for documentation
    - Print summary pages for safety committee meetings
    
    **Java version:**
    - Select **File → Save As** and choose a location
    - Use **Export → Generate Report** to create formatted output
    - Export to PDF, Word, or HTML format as needed
    
    <Note>
    Maintain version control of your assessments. BioRAM assessments should be:
    - Conducted annually or whenever significant changes occur
    - Stored securely with access controls
    - Reviewed by safety/security committees
    - Used to track improvements over time
    </Note>
  </Step>
</Steps>
